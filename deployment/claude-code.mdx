## Setup

Add your server to Claude Code's MCP configuration. In your project directory, create or edit `.mcp.json`:

```json .mcp.json
{
  "mcpServers": {
    "my-api": {
      "command": "python",
      "args": ["/absolute/path/to/my-api-server/server.py"]
    }
  }
}
```

Or use the generated `.mcp.json` file directly — it's pre-configured with the correct command.

### Using a virtual environment

Point to the Python binary inside your venv:

```json .mcp.json
{
  "mcpServers": {
    "my-api": {
      "command": "/absolute/path/to/my-api-server/.venv/bin/python",
      "args": ["/absolute/path/to/my-api-server/server.py"]
    }
  }
}
```

## Usage

Start Claude Code normally. Your API tools are available immediately. Ask Claude to perform operations:

```
> Use the my-api server to list all users
```

Claude Code will call the appropriate MCP tool and display the results.

## Scope

- **Project-level** `.mcp.json` — Tools available only in that project directory
- **Global** `~/.claude/.mcp.json` — Tools available in all projects
